Project Description:

  • RISC-V CPU-Design

    This is a RISC-V architecture control processing unit (CPU) design. The design was done through Logisim, a software that simulates digital logic circuits. The design utilizes built in components like counters, comparators, and more to achieve a cleaner design. The CPU design is capable of executing sorting algorithms, matrix multiplication, and other sequential instructions using Y86 (X86-like) assembly language.

  • Aggie Shell

    This is a personal simple designed linux shell that users can execute almost all linux command. The project utilzies the concepts of parent and child processes with file descriptors and pipes to execute long sequences of linux commands and not just one command at a time.

  • Text-Based RPG Game

    The text-based RPG game features three classes: Warriors, Magicians, and Assasins. Players can engage in battles against monsters or battle with random generated NPCs to gain experience and level up to unlock different skills. This game also has consumables to for players to restore status. In addition, there is an attribute system where players can add points to make their characters stronger. The game also includes a save and load feature, allowing players to preserve their progress and continue their adventure at a later time.
